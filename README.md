# Git-Native Model Registry — Glossary & Docs Portal

This repo contains a curated, enterprise-ready technical glossary for the **Git-Native Model Registry** product and a lightweight docs portal (MkDocs-ready).

Contents:
- `docs/` — Markdown docs for the portal
- `mkdocs.yml` — MkDocs configuration
- `README.md` — this file
- `glossary.csv` — machine-friendly CSV of the glossary
- `LICENSE` — MIT

Use this as the canonical documentation source for sales, engineering, legal, and onboarding.

## Quick start (locally)
1. Install MkDocs: `pip install mkdocs mkdocs-material`
2. Serve docs: `mkdocs serve`
3. Open `http://127.0.0.1:8000` in your browser.

## Repo usage
- Edit `docs/glossary.md` to update definitions.
- Contribute via PRs: new terms should include `Meaning`, `Spanish translation`, `Application`, `Examples`, `Synonyms`, `Environments`.
